## Usage

Don't. ... Yet.

## Installation

`git clone` this project into your quicklisp local-projects folder and quickload.

## Testing

Tests written with rove.

```
(asdf:test-system :collox)
```

Or if running rove from the CLI with roswell...

````
rove collox.asd
````
